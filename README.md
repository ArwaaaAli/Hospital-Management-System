# Hospital Management System – Data Structures Project

## Project Description
In today's fast-paced world, hospitals and healthcare systems require efficient and reliable tools 
to manage patient information, emergency cases, treatment history, and doctor scheduling. This 
project implements a **basic Hospital Management System** using core data structures learned in the course. By applying concepts such as **Linked Lists, Stacks, Priority Queues, and Hash Tables**, this project simulates a real-world hospital environment. It enhances Java coding skills and demonstrates practical applications of data structures.

## How to Run
1. Clone or download the project from GitHub:  
   [Hospital Management System Repository](https://github.com/ArwaaaAli/Hospital-Management-System)
2. Open the project in any Java IDE (e.g., Eclipse, IntelliJ)  
3. Run the **Main** class to test all functionalities:  
   - Patient management (Linked List)  
   - Emergency queue (Priority Queue)  
   - Treatment history (Stack)  
   - Doctor assignments (Hash Table)
